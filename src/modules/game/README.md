# Game Module

Capas del modulo:

- `domain`: tipos de compra de tablas, rondas, lineas ganadoras y premios.
- `application`: server actions y servicios para compras, motor de partidas y liquidacion de lineas.
- `infrastructure`: repositorio Supabase + RPCs transaccionales de juego y pagos.
- `ui`: formularios de compra/control admin, render de partidas/premios y sala realtime de juego.
